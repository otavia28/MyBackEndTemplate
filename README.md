## 介绍
基于 Java SpringBoot 的项目后端模板，整合了常用框架和一些常见业务。
## 业务
1. 用户注册。
2. 用户登录。
3. 管理员：查询用户。
4. 管理员：删除用户。
5. 查询当前用户。
6. 用户注销（退出登录）。
7. 用户信息更新。
## 框架
- Java 8
- SpringBoot 2.7.6
## 依赖
- MyBatis-Plus 3.5.3.2
- JUnit 4.13.2
- Apache Commons Lang 3.12.0
## 插件
- MyBatisX：自动生成代码文件
- GenerateAllSetter：自动生成所有 set 方法
- Auto Filling Java Call Arguments：自动填充方法所需要的参数
