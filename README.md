# 介绍
基于 Java SpringBoot 的项目后端模板，整合了常用框架和一些常见业务。

# 业务
1. 用户注册
2. 用户登录
3. 用户注销（退出登录）
4. 用户查询（by userName）（管理员）
5. 用户删除（by id）（管理员）
6. 获取当前登录用户
7. 修改用户信息（管理员）
8. 修改当前用户信息
9. 添加用户（管理员）

# 框架
- Java 8
- SpringBoot 2.7.6
